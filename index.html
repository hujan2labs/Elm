Introducing elm

Elm is an app that make user able to create content on the map --like drawing a route,
then chat it with others.

Elm's home is your local and followed chat about any events.
This chat has limited time so it always dynamic.

Elm is powered with Phantasm,
our original package that able to create any shape above any digital maps.
With this, Elm can be deployed in any country with their own local map.

Elm prototype is ready.
For now, it only internally shared and invitation only.

About Us

Hi, my name is Red. I'm solo developer based in Indonesia.

I started coding in 2011, and my first app is 'Where I Should Go Now' for iOS. My motivation came from frustration when using Google Maps routing. Its algorithm often gave me a road that's too narrow for a car to drive or too quiet and scary to walk. So, I built an app with the ability to save routing results (to choose) permanently. In 2012, I founded hujan2labs as a one-person company (Bentuk Usaha Perseorangan). Then, around 2015, I rebuilt the app for Android and named it 'Lungo' ('Going somewhere' in Javanese).

In 2020, I built elm. My motivation was because some new technologies became available. Flutter, Firebase, and enabled GitHub private repo for free were my motivations to develop again. This time, I built an app that allows users to draw their own routes and have conversations with others. I've also tried various methods with Phantasm, and now it can operate on any contemporary digital maps.

Now, I'm looking to build a startup with aim to make elm ready for global market.
2raindrops will be the new company name.
