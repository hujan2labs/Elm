Introducing elm

Elm is an app that make user able to create content on the map --like drawing a route,
then chat it with others.

Elm's home is your local and followed chat about any events.
This chat has limited time so it always dynamic.

Elm is powered with Phantasm,
our original package that able to create any shape above most digital maps.
With this, Elm can be deployed in most country with their own local map.

Elm prototype is ready.
For now, it only internally shared and invitation only.

About Us

Hi, my name is Red. I'm solo developer based in Indonesia.

I start coding in 2011, my first app is "Where I Should Go Now" for iOS. My motivation is from frustation when using Google Maps routing. It's algorithm often give me a road that's too narrow for car to drive or scary-quiet to walk. So I build an app with ability to save routing result (to choose) permanently. On 2012 I build hujan2labs as one person company (Bentuk Usaha Perseorangan), then around 2015 I rebuild the app for Android and name it "Lungo" ("Going somewhere" in Javanese).

In 2020 I build elm. My motivation is because some of new technology things become available. Flutter, Firebase, and enabled Github private repo for free are my motivation to develop again. This time I build an app that make user enable to draw it's own route and have conversation with others. I use updated method and knowledge that I've used before, like Phantasm.

Now, I'm looking to build a startup with aim to make elm ready for global market.
